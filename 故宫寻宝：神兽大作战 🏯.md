# 故宫寻宝：神兽大作战 🏯

## 项目介绍

这是一个专为三年级小朋友设计的**故宫定向越野互动网页游戏**。通过故事化的游戏设定，引导小朋友在虚拟故宫中探索五个主要景点，完成打卡任务，收集神兽徽章，最终找到失落的"九龙玉玺"。

## 游戏特色

✨ **沉浸式故事背景**
- 以"寻找失落的九龙玉玺"为主线，将打卡点串联成一个完整的冒险故事
- 每个打卡点都有独特的神兽守护和故事线索

🎭 **趣味打卡机制**
- 每个景点都要求小朋友做出特定的Pose打卡合照
- 完成打卡后获得相应的神兽徽章
- 集齐所有徽章后解锁终极奖励

🏆 **游戏化学习**
- 通过游戏方式学习故宫知识
- 每个景点都有适合三年级小朋友理解的知识介绍
- 通过徽章收集和进度条增强成就感

📱 **手机优化设计**
- 完全响应式设计，适配各种手机屏幕
- 大字体、大按钮，便于小朋友操作
- 流畅的动画和交互效果

## 游戏内容

### 五个打卡点

| 序号 | 景点名称 | 神兽徽章 | Pose要求 |
| :---: | :---: | :---: | :---: |
| 1 | 太和殿 (金銮殿) | 龙 | "我是皇帝"：双手叉腰，抬头挺胸 |
| 2 | 乾清宫 (皇帝寝宫) | 凤 | "正大光明"：指向牌匾，竖起大拇指 |
| 3 | 御花园 (皇家花园) | 狮 | "神兽守卫"：做出威武的蹲坐姿势 |
| 4 | 九龙壁 (彩色巨龙) | 海马 | "飞龙在天"：双臂张开，像龙飞翔 |
| 5 | 角楼 (故宫的眼睛) | 斗牛 | "瞭望远方"：手搭额头，眺望远方 |

### 故事线索

每完成一个打卡点，小朋友都会获得一条关于玉玺下落的线索：

1. "玉玺藏在**中轴线**上，它守护着**最高的宝座**。"
2. "玉玺在**皇帝休息**的地方，那里有**秘密的盒子**。"
3. "玉玺在**花园的中心**，被**最古老的树**看着。"
4. "玉玺在**九条龙**的旁边，它们**面朝北方**。"
5. "玉玺在**故宫的角落**，它能**看到整个皇宫**。"

### 终极奖励

集齐所有5枚神兽徽章后，小朋友将获得：
- 🎖️ "故宫小英雄"证书
- 🎁 精美的"九龙玉玺"图片
- 🏆 游戏完成的荣誉认可

## 使用方法

### 在线访问

1. 将项目文件上传到Web服务器
2. 在浏览器中访问 `index.html`
3. 在手机上打开链接，开始游戏

### 本地运行

1. 下载项目文件到本地
2. 用浏览器直接打开 `index.html`
3. 或者使用本地Web服务器运行（推荐用于测试）

```bash
# 使用Python简单HTTP服务器
python3 -m http.server 8000

# 然后在浏览器访问
http://localhost:8000/index.html
```
